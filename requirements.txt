# Jarvis Dependencies
# Install with: pip install -r requirements.txt

# -- Day 2 (voice + utilities) -------------------------------------------------
pyttsx3            # offline text-to-speech (no internet needed)
SpeechRecognition  # voice input via microphone
pyaudio            # microphone driver (required by SpeechRecognition)
                   # Windows: pip install pyaudio
                   # If pyaudio fails: pip install pipwin && pipwin install pyaudio

# -- Day 3 (automation + utilities) -------------------------------------------
psutil             # CPU / RAM / disk stats (sysinfo command)
requests           # HTTP calls (future: REST APIs)
python-dotenv      # load API keys from .env file

# -- Day 4 (scheduling + reminders) -------------------------------------------
# No new external dependencies -- uses only Python standard library
# (datetime, json, os, re) for the reminder engine.

# -- Day 5 (AI + personalization) ---------------------------------------------
# No mandatory new packages.  Install whichever AI provider you use:
# google-generativeai   # Gemini API  ->  pip install google-generativeai
# openai                # OpenAI API  ->  pip install openai
#
# If neither is installed, Jarvis falls back to the built-in rule-based AI.

# -- Day 6 (GUI + browser + logging) ------------------------------------------
# tkinter is built into Python (no pip needed — already included in Python 3.x)
# webbrowser is part of the Python standard library (no install needed)
# No new mandatory packages for Day 6.
#
# Optional: install pyinstaller to bundle Jarvis as a standalone .exe:
# pyinstaller          ->  pip install pyinstaller
